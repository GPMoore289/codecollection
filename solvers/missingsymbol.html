Missing Symbol Solver

Original Creation: 21st January 2023

Uploaded: 4th March 2025

Status: Complete. Original code was written without AI.

Description: This is an interesting illustration on how a computer solving a mathematical problem inefficiently can still be many orders of magnitude faster than a human. The question is - what mathematical operators must you insert into the equation 9 _ 2 _ 2 _ 8 = 16 in order for it to be correct? While a trivial problem for a computer to solve, it did introduce me to the eval function, which effectively converts strings into non-strings. My decision to randomize the insertion of each symbol rather than just cycle through them all systematically was not an informed one, but it still gets the result quickly.

    return v1 + v2

def sub(v1, v2):
    return v1 - v2

def mult(v1, v2):
    return v1 * v2

def div(v1, v2):
    return v1/v2

while True:
    ops = ["+", "-", "*", "/"]
    oporder = ["9", "x", "2", "x", "2", "x", "8"]
    v=0
    while v < 3:
        oporder[oporder.index("x")] = random.choice(ops)
        v += 1
    x = " ".join(oporder)
    print(x)
    z = eval(x)
    print(z)
    print("")
