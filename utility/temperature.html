Temperature Converter

Description: Easy program that converts Celsius to Fahrenheit or vice versa. This was as I was starting the Introduction to Programming course and I wanted to make something tightly scripted and brush up on basic error handling.

Original Creation: 11th Sep 2024

Uploaded: 22nd Mar 2025

Status: Complete. Original code was written without AI.

Supplemental: N/A

while True:
    entry = input("Please enter a temperature, ending with 'F' for Fahrenheit or 'C' for 'Celsius', or 'done' to exit:")
    if entry.lower() == "done":
        break
    if (entry.lower().endswith("f") != True) and (entry.lower().endswith("c") != True):
        print("Not a valid entry.\n")
        continue
    try:
        float(entry[0:-1])
    except:
        print("Not a valid entry.\n")
        continue
        
    if entry[-1].lower() == "f":
        fahrenheit = float(entry[0:-1])
        output = round((fahrenheit - 32) * (5/9), 2)
        print(f"{fahrenheit} *F is {output} *C\n")

    elif entry[-1].lower() == "c":
        celsius = float(entry[0:-1])
        output = round(((celsius * 1.8) + 32), 2)
        print(f"{celsius} *C is {output} *F\n")
