Paper-Scissors-Rock

Original Creation: 29th April 2020

Uploaded: 25th Feb 2025

Status: Complete. Original code was written without AI.

Description: Every coder has to make this at least once, right? This was early in my coding learning, and my choice of names for variables is... not easy to read. And for some reason, I thought putting the import function in the middle of the while loop was a good idea... that's actually kind of adorable in hindsight. Still, I think this was the first time I used the def function to shorten the code and follow the "don't repeat yourself" mantra.

bank = ["P", "S", "R", "Q"]
def fn1(v1, v2, v3, v4):
	if v1 == v2:
		print("You" + v4 + " chose " + v3)
while True:
	while True:
		ent = input("Enter P (for paper), S (for scissors), R (for rock) or Q to quit: ")
		if not ent in bank:
			print("Please enter a valid letter")
			continue
		else:
			break
	if ent == "Q":
		break
	fn1(ent, "P", "Paper", "")
	fn1(ent, "S", "Scissors", "")
	fn1(ent, "R", "Rock", "")
	import random
	while True:
		a = random.choice(bank)
		if a == "Q":
			continue
		else:
			break
	fn1(a, "P", "Paper", "r opponent")
	fn1(a, "S", "Scissors", "r opponent")
	fn1(a, "R", "Rock", "r opponent")
	if ent == a:
		print("Draw\n")
		continue
	if (ent == "P" and a == "S") or (ent == "S" and a == "R") or (ent == "R" and a == "P"):
		print("You lose\n")
		continue
	if (a == "P" and ent == "S") or (a == "S" and ent == "R") or (a == "R" and ent == "P"):
		print("You win\n")
		continue
