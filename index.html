Basic Programming Projects

These were created from September to December of 2024, as part of the Programming Fundamentals course at Mohawk College. I already had a good grip of Python by this point, so solving these problems were often trivial in most cases. However, the course did introduce me to the use of AI in programming, and placed an emphasis on correct code-writing discipline, as well as identifying errors in code.

Age & Gender: Output average ages for each gender based on CSV data.

Birthday Problem: Figure out the probability of X people sharing a birthday.

Change Finder: Figure out the minimum coins needed to make an amount of money.

Favourite Films: A very simple program for adding or removing items from a list.

FizzBuzz: The classic programming challenge - print Fizz, Buzz or FizzBuzz at the correct moment.

Inventory Manager: Add items, prices and quantities to a list.

Palindrome Finder: Detects if a sentence is a palindrome.

Password Checker: Asks for password creation and accepts/rejects based on strict criteria.

Sentence Deconstructor: Summarizes all characters used in a sentence.

Word List Filter: Add words to a list, then remove elements based on word length.

Math Challenges

Sometimes, I'll come across a math problem during my internet perusal, and see if I can solve it using Python.

Birthday Calculator: Determines what day of the week you were born on.

Electoral College Calculator: Allocates Electoral College votes to US states based on population.

Fibonacci Calculator: Enter a number, program will print that many numbers into the Fibonacci sequence.

Frog Jump Problem: Determines average number of jumps a frog will make across 10 lilypads.

Golden Ratio Selection: How many items should you inspect before making a decision?

McNugget Problem Solver: Calculates the maximum number of Chicken McNuggets that can't be bought from McDonalds.

Prime Finder: Calculates if a number is prime, or its divisors if it isn't.

Utility Projects

A bit more of a mish-mash of programming challenges, but these are intended to have a more real-life (albeit niche) applications.

Binary Converters: Convert to and from binary.

Family Tree (incomplete): Program that helps me navigate my partner's family tree!

Film Rankings: Short program that lets me quickly number a list of films in a text document.

Password Cracker: Demonstration of how brute force cracking is effective with weak passwords.

Temperature Converter: Convert Celsius to Fahrenheit and vice versa.

Terraforming Mars Web Scraper: Collects my game data from an online board game website.

Game Solvers

Similar to the Math Challenges, but focused on solving game problems (mobile, video game or even board games) using code.

D&D Dice Roller: A simple way of solving a D&D dice-rolling conundrum.

Final Fantasy IX Skipping & Running: Two "bots" that automate Final Fantasy IX's two most awkward challenges.

Infinitode Bounty Solver: What's the optimal strategy for buying money-making towers in a tower defense game?

Missing Symbol Solver: Find the missing mathematical symbols to make an equation work.

Water Sort Puz Solver: Find a solution to a water-sorting puzzle game in seconds.
