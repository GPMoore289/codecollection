McNugget Problem Solver

Description: This is a silly one, but a useful exercise in chaining "for" statements. The McNugget problem ponders what is the maximum value of Chicken McNuggets you can't buy from McDonalds, given they are sold in packs of 6, 9 and 20. This is a simple brute force solution that will give the correct answer of 43 (at least for values less than 1000).

n = 167
allnums = {6, 9, 20}
uniquenums = []

while n > 0:
    n -= 1
    newnums = set()
    elems = len(allnums)
    for i in allnums:
        for j in allnums:
            x = i + j
            if x < 1000:
                newnums.add(x)
    for i in newnums:
        allnums.add(i)
        
for i in range(0, 1000):
    if i not in allnums:
        uniquenums.append(i)
        
print(max(uniquenums))
