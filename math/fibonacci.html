Description: Fibonacci calculators are a go-to practice exercise for new coders. While the idea for this came about as I was learning about function recursion, I preferred the recursion-free approach as it prints results far faster. This also introduced me to the yield function.

Calculator will print this many numbers in the Fibonacci sequence

	def fibonacci(n):
		count = 0
		a = 1
		b = 0
		while count < n:
			count +=1
			if count == 1:
				yield b
				continue
			if count == 2:
				yield a
				continue
			if count > 2:
				c = a + b
				yield c
				if count%2 == 1:
					b = c
				if count%2 == 0:
					a = c
				continue

	n = int(input("Enter a number:"))
