Prime Finder

Original Creation: 8th August 2024

Uploaded: 25th Feb 2025

Status: Complete. Original code was written without AI.

Description: This was a simple exercise in optimising run time. This program determines whether a number is prime, or if it isn't, outputs its divisors. Realizing that you don't have to search beyond the square root of the number sped up calculation speed by orders of magnitude.

import math

while True:

	strquery = input("Enter number: ")
	try: 
		int(strquery)
	except:
		print("Please enter an integer")
		continue
	
	query = int(strquery)
	divbank = []
	limit = int(round(math.sqrt(query)))
	n = 2
	
	while n <= limit:
		remainder = query%n
		if remainder == 0:
			divbank.append(n)
			divbank.append(int(query/n))
		n += 1
			
	if len(divbank) == 0:
		print(f"{query} has no divisors. It is prime.")
	else:
		print(f"{query} has the following divisors. It is not prime.")
		divbank.sort()
		print(divbank)
