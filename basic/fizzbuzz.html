FizzBuzz

Description: Every programmer has to attempt this at some point. This will mimic two players playing FizzBuzz: you alternate between players counting from 1 to 100, unless the number is divisible by 3 or 5, in which case, they say "Fizz" or "Buzz" respectively, unless the number is divisible by 15, in which case, they say "FizzBuzz". Nothing too challenging other than considering optimal arrangements of your if/elif/else statements.

n=0
player = ""
word = ""
divis1 = 3
divis2 = 5
while n <= 100:
    n+=1
    if n%2 == 0:
        player = 2
    else:
        player = 1
    
    if n%divis2 == 0 and n%divis1 == 0:
        word = "FizzBuzz"
    elif n%divis1 == 0:
        word = "Fizz"
    elif n%divis2 == 0:
        word = "Buzz"
    else:
        word = n

    print(f"Player {player} says {word}")
