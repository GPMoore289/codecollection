System Report Generator

Description: Being a Unix admin means knowing how well the system is running. We were tasked with writing a script which would generate information such as how long the system has been running, number of tasks running, etc. Most of these come directly from bash commands, but getting the CPU information into a neat sentence was a little trickier.

Original Creation: 29th Mar 2025

Uploaded: 14th Jun 2025

Status: Complete. Original code was written without AI.

Supplemental: Text: script run instructions

#!/bin/bash
#Graham Moore, 000957294, 29th March 2025

#I'm aware the first two commands will work without the echo statement, but I wanted them to always print on a new line
echo -e "\n$(uptime)"
echo -e "\n$(df -h)"
echo -e "\nThere are $(rpm -qa | wc -l) system packages installed."
echo -e "\nSystem host name: $(hostname)"
echo -e "\nYour CPU is a(n) $(lscpu | grep "Model name" | cut -d : -f 2 | awk '$1=$1') running at $(lscpu | grep "CPU MHz" | cut -d : -f 2 | awk '$1=$1') MHz."
#I got the idea for using the awk command from a forum post on StackExchange, asked by annahri in 2020. This was needed to remove the extra spaces after the colon.
echo -e "\n$(top -b -n 1| grep "Tasks")\n" #need -b for batch mode so it can be pipelined in plain text. Need -n 1 so that it stops after the first cycle.
