Word Filter

Description: Simple project: you start with a list of words that you can add extra words to. Once done, type an integer, and the word list will be filtered to only include words of that length or shorter. While mostly straightforward, this did alert me of the dangers of removing items from a list that is currently being iterated, and it's generally way safer to make a copy list first.

wordlist = ["apple", "banana", "avocado", "peach", "coconut", "persimmon", "plum", "orange", "melon", "guava", "cherry"]

while True:
    word = input("Please enter a word, or type 'done':").lower()

    if word == "done":
        break
    else:
        wordlist.append(word)

while True:
    length = input("Please enter the word character limit:")

    try:
        int(length)
    except:
        print("Please enter an integer.")
        continue
    else:
        break

length = int(length)

removelist = []
for word in wordlist:
    print(word)
    if len(word) > length:
        removelist.append(word)

for word in removelist:
    wordlist.remove(word)

print("")
for word in wordlist:
    print(word)
