Change Finder

Original Creation: 2nd October 2024

Uploaded: 20th Mar 2025

Status: Complete. Original code was written without AI.

Description:Enter an amount of money, and the program will calculate the smallest number of coins needed to create that amount of money. While Canada stopped minting pennies in 2013, they are technically still legal currency as of 2025, though a shop won't give you them in change. The calculation here was straightforward: continuously remove the highest denomination possible, though the round function was necessary to remove floating point errors.

while True:
    changeentry = input("Enter amount of change:")

    try:
        float(changeentry)
    except:
        print("Enter a float or integer")
        continue

    change = float(changeentry)
    toonies = 0
    loonies = 0
    quarters = 0
    dimes = 0
    nickels = 0
    pennies = 0

    while change >= 2.00:
       change = round(change - 2.00, 2)
       toonies += 1
    while change >= 1.00:
       change = round(change - 1.00, 2)
       loonies += 1       
    while change >= 0.25:
       change = round(change - 0.25, 2)
       quarters += 1
    while change >= 0.10:
       change = round(change - 0.10, 2)
       dimes += 1
    while change >= 0.05:
      change = round(change - 0.05, 2)
      nickels += 1
    while change >= 0.01:
      change = round(change - 0.01, 2)
      pennies += 1

    print(f"You can make {changeentry} with {toonies} toonies, {loonies} loonies, {quarters} quarters, {dimes} dimes, {nickels} nickels and {pennies} pennies.")
