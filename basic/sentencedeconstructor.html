Sentence Deconstructor

Status: Complete. Original code was written without AI.

Description: Enter a phrase, letters and spaces only, and the program will count how many of each letter was used. This was a quick in-class creation that used a dictionary to expedite the code, though I wish I had remembered to include a ".lower()" correction in the input.

alphabet = {'a' : 0,'b' : 0, 'c' : 0, 'd' : 0, 'e' : 0, 'f' : 0, 'g' : 0, 'h' : 0, 'i' : 0, 'j' : 0, 'k' : 0, 'l' : 0, 'm' : 0, 'n' : 0, 'o' : 0, 'p' : 0, 'q' : 0, 'r' : 0, 's' : 0, 't' : 0, 'u' : 0, 'v' : 0, 'w' : 0, 'x' : 0, 'y' : 0, 'z' : 0, " " : 0}

while True:
    sentence = input("Please enter a sentence containing only letters and spaces:")

    for letter in sentence:
        if letter not in alphabet.keys():
            print("This sentence contains invalid character.")
            continue
        else:
            value = alphabet[letter]
            value += 1
            alphabet[letter] = value

    for key, value in alphabet.items():
        if key != " ":
            print(f"{key}: {value}")
            alphabet[key] = 0
        else:
            print(f"spaces: {value}")
            alphabet[key] = 0
