Palindrome Finder

Original Creation: 3rd October 2024

Uploaded: 19th Mar 2025

Status: Complete. Original code was written without AI.

Description: Enter a sentence, program will detect if it is a palindrome or not. For such short code, the backwards string scanning calculation is not immediately intuitive, but what works, works.

while True:
    actualentry = input("Enter a word or sentence:")

    entry = actualentry.lower()
    entry = entry.replace(".", "")
    entry = entry.replace(",", "")
    entry = entry.replace(" ", "")
    entry = entry.replace("'", "")
    
    palindrome = True
    length = len(entry)
    n = 0
    
    while n < length:
        if entry[n] == entry[-1*n -1]:
            n += 1
            continue
        if entry[n] != entry[-1*n -1]:
            palindrome = False
            break

    if palindrome == True:
        print (f"{actualentry} is a palindrome.\n")
    else:
        print (f"{actualentry} is not a palindrome.\n")
