Password Checker

Original Creation: 12th September 2024

Uploaded: 25th Mar 2025

Status: Complete. Original code was written without AI.

Description: This program emphasizes the importance of strictly controlling input. The user is asked to enter a password that meets the criteria and will "fail" the user unless they do everything correctly. Maybe I should see if my password cracker can break this one?

uppercase = "ABCDEFGHIJKLMNOPQRSTUVWXYZ" #bank of all uppercase characters
lowercase = "abcdefghijklmnopqrstuvwxyz" #bank of all lowercase characters
digits = "0123456789" #bank of all numerical digits

while True: # creates the program loop
    entry = input("Please enter a password. Password must be at least 8 characters long, contain a mix of uppercase and lowercase letters, and at least one number: ") #user enters input
    UCheck, LCheck, DCheck = False, False, False #program is working on a "guilty until proven innocent" logic: these variables are set to False and must switch to True before password can be accepted
   
   if len(entry) < 8: #checks length of password. If it is less than 8, it will trigger if statement
        print("Password is too short, must be at least 8 characters long.\n") #alerts user if too short, then is sent back to start of loop
        continue
    for i in entry: #for loop that checks each character against each of the banks. 
        if i in uppercase: #As soon as an uppercase letter is detected, the UCheck variable is set to true.
            UCheck = True
        if i in lowercase: #As soon as a lowercase letter is detected, the LCheck variable is set to true.
            LCheck = True
        if i in digits: #As soon as digit is detected, the DCheck variable is set to true.
            DCheck = True

    if UCheck == False: #If the UCheck variable is still False, alerts user to include at least one uppercase letter, then sends them back to start of the loop.
        print("Password must contain at least one uppercase letter.\n")
        continue
    if LCheck == False: #If the LCheck variable is still False, alerts user to include at least one lowercase letter, then sends them back to start of the loop.
        print("Password must contain at least one lowercase letter.\n")
        continue
    if DCheck == False: #If the DCheck variable is still False, alerts user to include at least one digit, then sends them back to start of the loop.
        print("Password must contain at least one digit.\n")
        continue
    else: #user can only have reached this point if all Check variables are True and length is greater than 8. Ergo, password is accepted.
        print("Password meets criteria.\n")
        while True: #loop for ensuring the user enters the same password again.
            entry2 = input("Please enter the same password again to confirm. If you want to restart the password entry process, enter 'restart': ")
            if entry2.lower() == "restart": #breaks out of loop if user enters "restart". Will then reach the if statement outside of this loop and get sent back to the start of the main loop.
                break
            if entry2 != entry: #checks if second password matches the first. Sends user back to the start of the loop otherwise.
                print("Does not match original password.\n")
                continue
            else:
                break
        if entry2 == "restart": #sends user back to the start of the main loop if they want to restart the password creation process
            continue
        print("Password accepted!")
        break
